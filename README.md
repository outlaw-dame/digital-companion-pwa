# Digital Companion PWA

A digital partner that evolves based on your habits, combining the grace of a White Tiger and the mythical architecture of a Small Dragon.

## Tech Stack

- **Runtime**: Bun
- **Frontend**: Vite + React 19
- **UI Components**: Konsta UI v5
- **Styling**: Tailwind v4
- **PWA**: vite-plugin-pwa
- **Backend**: Bun HTTP server
- **Storage**: bun:sqlite
- **Platform Detection**: navigator.userAgent + CSS (env)

## Project Structure

- `/frontend`: Vite + React application
- `/backend`: Bun HTTP server with SQLite

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
bun run index.ts
```

## Features

- **Saint-Beast Hybrid**: A unique digital companion with high EQ and intelligence.
- **Real-time Debugger**: Acts as a sanity check for your ideas.
- **Safety Protocol**: Stabilizing presence during high stress.
- **PWA Support**: Installable on mobile and desktop.
